2023-05-08考研数学规划

发布于 2023-05-08
@1xiao

建议顺序

高数基础——>现代基础——>高数强化——>现代强化——>概率基础强化二合一的顺序。这样的好处是万一学不完往数二转也可以。

基础课可听可不听。

大概需要50天学习高数和现代的强化。强化课必须要听，站在巨人的肩膀上学习，只会事半功倍。

以8个月为例子

  • 第1~3月基础，第4~6月强化，第7~8月冲刺（12月背政治）；
  • 3月到9月一天建议5h以上，建议早上、晚上学数学

原因很简单，数学上午考，且早上思维活跃

早上雷打不动用来做题，晚上看视频课学习

做题相比于看视频课更为痛苦，越不想做的事情越要放到前面。
数学能力的提高是靠一张张草稿纸堆出来的。

下午就可以学一点别的东西，比如英语和专业课。

10月以后政治英语会占用大量时间，数学每天3~4小时即可。

资料&视频

建议跟谁的视频就用谁的书。

建议的话，老师

  • 高数：基础阶段可以武忠祥或张宇，强化阶段武忠祥
  • 现代：李永乐or张宇or汤家凤
  • 概率：基础阶段建议余丙森，强化阶段跟余丙森or王世安

习题

  • 基础阶段：张宇300or汤1800（不推
  • 强化阶段李永乐660 和 李林880
  • 冲刺阶段利用了和李艳芳 和 模拟题李林6 李林4

和 表示都必做

真题大概需要1个月，建议放在10月，可以早但不可以晚。模拟题不要停，锻炼考场思维

巧妙快速的做出选择题，找到做题顺序等等。

==建议补充武忠祥17堂课==

复盘

能力的提高不在于做了多少题，而在于做了多少总结

复盘什么

  1. 计算出问题的地方
  2. 思路卡壳、解题不顺畅的地方
  3. 易错点、细节类问题的归纳
  4. 专题总结
  5. 总结解题框架
  6. 知识框架和总结题型
  7. 难题

重点1、2、3；次重点4、5；

注意事项

  1. 做题为主，视频为辅
  2. 做题时别玩手机，别翻答案，训练连续三个小时做题的能力，考场上你会更感谢自己
  3. 少做无用功感动自己，浪费时间。标记好错题的位置，重要的是多翻多做错题
