2023，我的职业规划

发布于 2023-09-20
@xiao

引言

  • 试错的成本并不高，而错过的成本非常高。——雷军
  • 选择只是一瞬间的事，剩下的时间用来还债。

写这些话是想表达，选择不同的职业，注定走向不同的道路，我还未毕业，还有时间思考未来，相比于未来几十年的职业生涯，几个月的时间简直不值一提。

前言

我是1xiao，这篇blog讲述了我在大学临近毕业的时候，对自己将来人生的职业思考。

应届毕业生选择的路无非三条：考研、就业、考公。可是大多数人从来没有一个明确的职业规划，我（9月以前）就没有。好在现在有了。

2023年5月下旬，我很迷茫，即使每天都在为考研做准备，但还是和中铁十一局签了《就业意向书》。

2023年7月，还是没有目标，我为何而战？我离开学校，回了家……

2023年9月，三方协议向我发起邀约，我确定要进入土木，当牛马了吗？明明大家都知道这个行业的惨淡。

我必须思考自己的未来……

我的思路

毕业后，假如我进了土木，各种因素可能是这样的👇

  1. 地理位置：施工单位需要跟着项目走，中铁的项目铁路、地铁居多，都在人烟稀少之地；
  2. 工作环境：露天作业，风吹日晒，看看B站的大猛子，被摧残的像个40岁的中年人；
  3. 人脉圈子：职业发展前几年要跟着项目走，平日里接触的是农民工，就算5年后升值了，接触到的也只是土木领域的人；
  4. 职业前途：传统行业是旧的既得利益者已制定好的系统，关系户较多，职业发展难；
  5. 工作强度：风吹日晒非常消磨精力，学习？几乎不可能。晚上学？赶工期，做梦；
  6. 技能积累：学到的本领除了用于土木领域，无法迁移到其他行业；
  7. 社交氛围：接触的人文化程度偏低、年龄较大，有代沟，容易与社会脱节。

那如果不去土木，我想要什么呢？

  1. 我想要工作之余，有一个能够阅读的环境；
  2. 我要做不确定性更大的事，不要走已知晓答案的职业路线（有种认命的绝望感）；
  3. 做有益于社会的事，我希望能得到及时的反馈；
  4. 我认为活着是为了创造，本科生进入土木行业，没什么可研究、创造的，只有重复；
  5. 我希望我的工作是一场无限游戏，能够让我积累“受益终生”的技能；
  6. 我希望我能和年轻人一起工作，而不是走在社会边缘，过时落伍；
  7. 我希望我的工作能和 OpenAI 类工具相关，谁都知道它是未来的钥匙；
  8. 我希望我能有周末假期，去参加交流会、培训课，去认识更多的人，土木行业不可能满足这一条件；
  9. 我希望我能在一线城市工作，那里的机会多。

综上，传统模式的土木行业无法满足我的要求，就算第一份工作是国企，我也早晚要跳槽。既然早晚要跳槽，倒不如第一份工作就跳槽。

2023-09-20 晚21:42，已写50+min，剩下的明天写。

后来我发现有一种职业叫做“新媒体运营”，这种职业不仅是互联网领域的，还不限制求职者的专业，目前看来，这是进入互联网行业的最佳路径。

假如我进入新媒体运营行业，好的地方可能是这样的👇

  1. 地理位置：互联网公司大多在一线、新一线城市，人比较多；
  2. 工作环境：室内工作，不会因为温度影响效率；
  3. 人脉圈子：接触的年轻人居多，且有可能在工作之余认识其他行业的人，对职业发展有利；
  4. 职业前途：任何产品都需要营销，运营就是新时代的线上营销，还可以向自媒体等方向发展；
  5. 工作强度：可能会稍累，但是刚入职场，最重要的是学到真本领；
  6. 技能积累：自媒体技能，写作技能，搞得好还能学到一些商业逻辑；
  7. 社交氛围：接触的人几乎都受过教育（虽然可能没什么用），总体还行。

当然了，任何一个行业都有优缺点，但是在选择职业时，要选择自己认同的职业，反之，原理自己不认同的职业。否则每天花大量精力用于对抗不满的环境，是对年轻最大的浪费。

题外话，运营不看重专业，我认为可能的原因有：1.教育体系没有对口的专业教这方面技能；2.这份工作的行业标准仍未建立，工作内容混而杂；3.互联网领域更看重能力。
也正因为它不看重专业，竞争压力一定会有，但是，我想去。

一些事

来我们这儿秋招的中建、中交、中铁，除了最顶尖的2~3个公司，其他各公司对应届生的能力几乎无要求，公告上只是写写做样子而已，真正面试时，学生的简历内容也用不着分辨真假，只要是个男的，大多数公司的门就为你敞开。

  • 我们班有个同学，简历上没有突出的地方，自我介绍是“喜欢唱跳”，面试时被问了一道两个两位数相加的小学数学题，就通过了面试。

  • 还有一个男同学，挂过不只一科，绩点极低；另一个女同学，绩点极高，且拿过许多奖项。两人一起面试，公司要了男生，拒了女生。土木的性别歧视是存在的。

我看过一些被录取者的简历，优秀的并不多。

我在决定新媒体运营前很犹豫，班主任多次劝我，说这是个好工作，可我问他“十一局怎么样”，他的回答最初是“我也不了解这个公司”，后来是“可能是个好公司”，最后是“肯定不错”。

班上许多同学也会请教班主任“某某局是不是好公司”这样的问题，他的回答统一是：“好好，都是好公司”。

真的是这样吗？我疑心很重。没有任何“坏公司”，那“好公司”又怎样体现的出来？好和坏难道不是通过对比才有的吗？

总总迹象令我怀疑自己当初签约是一个错误的决定。若是进入一个门槛如此低的行业，会有什么前途呢？

所以，我决定放弃秋招的机会，专心学习运营所需的技能，明年再找工作。
